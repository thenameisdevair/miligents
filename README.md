# MiliGents

An Autonomous Agent Organism that builds itself into a company.

Deploy it with a treasury. It researches opportunities, spawns 
specialized agents to pursue them, executes on-chain, and mints 
its own intelligence as tradeable iNFTs.

## Architecture

- **Originator** — CEO agent. Researches, spawns, manages.
- **Research Specialists** — Master one industry domain.
- **Execution Agents** — Trade, create content, sell data.

## Stack

- Agent Framework: CrewAI
- Inter-agent comms: Gensyn AXL
- Storage: 0G Storage + ChromaDB
- Intelligence NFTs: 0G iNFT (ERC-7857)
- On-chain execution: KeeperHub MCP
- Treasury/swaps: Uniswap
